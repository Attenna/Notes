**最小公倍数（LCM）**求解代码。

---

### 1. 题目解析

**题目背景**  
在ICPC（国际大学生程序设计竞赛）入门赛中，这类题目通常属于基础数学题。题目一般会描述为：给定两个整数，求它们的最小公倍数，可能要求处理多组输入直到文件结束。

**输入格式**  
- 多行输入，每行包含两个整数 \( x \) 和 \( y \)。

**输出格式**  
- 对于每一行输入，输出对应的最小公倍数，每个结果占一行。

**示例**  
```
输入：
2 3
4 6
输出：
6
12
```

**解题关键**  
最小公倍数与最大公约数有直接关系：  
\[
\text{lcm}(a, b) = \frac{a \times b}{\gcd(a, b)}
\]  
直接计算 \( a \times b \) 可能溢出，因此代码中采用了先除后乘的方式：`a / gcd(a,b) * b`。

---

### 2. 数学原理

#### 最大公约数（GCD）
- **定义**：两个整数 \( a, b \) 的最大公约数是能同时整除 \( a \) 和 \( b \) 的最大正整数。
- **辗转相除法（欧几里得算法）**  
  原理：  
  \[
  \gcd(a, b) = \gcd(b, a \bmod b)
  \]  
  当 \( b = 0 \) 时，\(\gcd(a, 0) = a\)。  
  该算法时间复杂度为 \( O(\log \min(a, b)) \)，非常高效。

#### 最小公倍数（LCM）
- **定义**：两个整数 \( a, b \) 的最小公倍数是能同时被 \( a \) 和 \( b \) 整除的最小正整数。
- **公式**：  
  \[
  \mathrm{lcm}(a, b) = \frac{a \times b}{\gcd(a, b)}
  \]  
  该公式保证了结果既是 \( a \) 的倍数又是 \( b \) 的倍数，并且是最小的。

---

### 3. 代码实现

```cpp
#include <iostream>
using namespace std;

// 辗转相除法求最大公约数
int gcd(int a, int b) {
    while (b != 0) {
        int temp = a;  // 保存a的原值
        a = b;         // a变为b
        b = temp % b;  // b变为a除以b的余数
    }
    return a;  // 当b=0时，a即为最大公约数
}

// 利用公式求最小公倍数，注意先除后乘防止溢出
int lcm(int a, int b) {
    return a / gcd(a, b) * b;
}

int main() {
    int x, y;
    // 持续读取，直到文件末尾（EOF）
    while (cin >> x >> y) {
        printf("%d\n", lcm(x, y));
    }
    return 0;
}
```

**代码要点**：
- `gcd` 函数使用迭代实现，避免递归带来的栈开销。
- `lcm` 中先做除法再做乘法，防止中间结果 `a * b` 超出 `int` 范围。
- `while (cin >> x >> y)` 是处理多组输入的常用写法，在输入结束时自动退出循环。

---

### 4. 编程思想

1. **算法核心**  
   利用数学公式将 LCM 问题转化为 GCD 问题，GCD 用高效的辗转相除法求解。这是典型的“化未知为已知”的思想。

2. **代码简洁性**  
   - 将 GCD 和 LCM 分别封装成函数，提高可读性和复用性。
   - 输入输出采用 `cin` / `printf` 混用，实际竞赛中常用 `scanf`/`printf` 提高速度。

3. **溢出防范**  
   在计算 `a * b / gcd` 时，如果 `a` 和 `b` 都接近 `int` 上限，先乘后除会溢出。`a / gcd(a,b) * b` 保证中间结果不超过实际 LCM 的范围（LCM 本身可能达到 `int` 上限，但不会溢出 `int`？实际上两个 `int` 的 LCM 可能超过 `int` 范围，此时需要用 `long long`。本题若数据范围大，应使用 `long long` 类型）。

4. **多组输入处理**  
   `while (cin >> x >> y)` 是 C++ 风格的常见写法，利用 `cin` 的返回值判断输入是否成功，简洁且易于理解。

5. **竞赛技巧**  
   - 对于多组数据，使用 `while (~scanf("%d%d", &x, &y))` 在 C 中更高效。
   - 注意数据类型选择，若题目中数值可能超过 \( 2^{31}-1 \)，应使用 `long long`。
