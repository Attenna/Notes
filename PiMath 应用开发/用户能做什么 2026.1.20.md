### 一、MVP 阶段最终可交付的核心成果

基于这套分工和方案，最终能做出一个**跨平台、Web 界面操作、轻量化的仓颉版极简 MATLAB 工具**，核心能力如下（全部可落地，无复杂技术壁垒）：

1. 核心：支持 MATLAB 风格的简洁脚本输入，完成基础矩阵 / 标量计算 + 折线图 / 柱状图可视化；
2. 辅助：能记录操作日志，支持 “回滚到上一步” 的版本回溯；
3. 体验：全 Web 界面操作（不用命令行），错误提示友好，跨 Windows/Linux/macOS 运行；
4. 边界：不支持复杂 3D 绘图、插件扩展、大规模并行计算，仅满足 “日常小规模科学计算” 需求（比如学生做线性代数作业、简单数据可视化）。

### 二、用户操作全流程（怎么输→输什么→输出什么）

#### 1. 用户 “怎么输入”（操作步骤，极简无门槛）

全程在**浏览器 Web 界面**操作，不用碰命令行 / 代码编辑器，步骤只有 3 步：

| 步骤  | 具体操作                            | 界面反馈                                                 |
| --- | ------------------------------- | ---------------------------------------------------- |
| 1   | 双击启动程序（或执行`cj run src/main.cj`） | 自动弹出浏览器窗口，加载 Web 界面（包含 3 个区域：脚本编辑区、工作区展示区、运行 / 回滚按钮） |
| 2   | 在左侧 “脚本编辑区” 输入 / 粘贴脚本，脚本语言是仓颉   | 编辑区有仓颉语法高亮（比如关键字标蓝、数字标绿），支持换行 / 删除等基础编辑              |
| 3   | 点击界面上的 “运行脚本” 按钮（或按 Ctrl+R）     | 按钮变灰，短暂加载后显示结果；若有错误，底部提示区显示红色错误文字                    |

#### 2. 用户 “输入什么”（支持的脚本类型，带 MATLAB 风格语法糖）

用户输入的是**带 MATLAB 语法糖的极简仓颉脚本**（不用写复杂的`import`/ 类型定义），核心支持以下两类输入，举 3 个典型示例：

##### 示例 1：基础标量 / 矩阵计算（带语法糖）

```cangjie
// 用户输入（MATLAB风格简洁写法，不用写pimath.前缀）
x=1:10;        // 语法糖：等价于x = pimath.linspace(1,10,10)
y=2*x + 5;     // 标量乘法+加法，自动调用pimath的矩阵运算
z=matMul(y,y); // 矩阵乘法，直接调用封装后的pimath函数
```

##### 示例 2：计算 + 可视化（核心场景）

```cangjie
// 用户输入
t=0:0.1:10;    // 语法糖：0到10，步长0.1生成数组
s=sin(t);      // 正弦函数，调用pimath的sin
plot(t,s);     // 语法糖：等价于simplePlot(t,s)，绘制折线图
```

##### 示例 3：错误输入（测试容错）

```cangjie
// 用户输入（故意写错，测试错误提示）
a=1:5;
plot(a,b);     // b未定义
```

#### 3. 程序 “输出什么”（对应输入的反馈，分 4 类）

所有输出都在 Web 界面 / 弹窗中呈现，无后台日志，用户能直观看到：

|输出类型|具体表现|对应示例|
|---|---|---|
|① 工作区变量展示|Web 界面右侧 “工作区展示区” 以表格形式显示：变量名、类型、维度 / 值|输入示例 1 后，表格显示：<br><br>x: Matrix [10x1]<br><br>y: Matrix [10x1]<br><br>z: Matrix [10x10]|
|② 绘图结果|自动弹出 gnuplot 窗口，显示绘制的图表（折线图 / 柱状图）|输入示例 2 后，弹出窗口显示 “t-s 的正弦曲线”，支持缩放 / 关闭|
|③ 版本回溯输出|点击 “回滚上一步” 按钮，工作区清空最后一步操作的变量，表格同步更新|输入示例 1 后点击回滚，z 变量消失，表格只剩 x、y|
|④ 错误提示|Web 界面底部红色文字提示，定位错误原因（人话描述）|输入示例 3 后，提示：“变量 b 未定义于工作区，请先定义 b 后再执行 plot”|

### 三、端到端典型使用示例（用户视角完整流程）

1. 用户双击启动程序 → 浏览器打开 Web 界面；
2. 在编辑区输入：
    ```cangjie
    t=0:0.05:5;
    y1=sin(t);
    y2=cos(t);
    plot(t,y1);
    ```
3. 点击 “运行脚本” → 右侧工作区显示`t: Matrix [101x1]`、`y1: Matrix [101x1]`、`y2: Matrix [101x1]`；
4. 自动弹出 gnuplot 窗口，显示`t-y1`的正弦曲线；
5. 用户点击 “回滚上一步” → 工作区中`y2`消失（最后一步是 y2 的赋值），绘图窗口保持不变；
6. 用户修改脚本为`plot(t,y2)`（此时 y2 已被回滚删除），点击运行 → 底部提示 “变量 y2 未定义于工作区”。
