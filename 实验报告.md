### Functional analyze:
The core function of this circuit is to perform integral operation on the input signal $u_i$. The output voltage $u_o$ is proportional to the integral of the input signal, with the formula: $$u_o = -\frac{1}{R_1 C} \int u_i dt$$When switch K is closed: The capacitor is short-circuited, there is no integral effect in the feedback loop, and the output $u_o$ remains stable (no integral change);
 When switch K is opened: The capacitor starts to accumulate charge, the input signal $u_i$ charges the capacitor through $R_1$, and the output $u_o$ changes linearly with the integral of the input signal (e.g., the output is a ramp wave under DC input).

### circuit diagram
![[Pasted image 20251217225111.png]]
### Functional analyze:
The core function of this circuit is to perform differential operation on the input signal $u_i$. The output voltage $u_o$ is proportional to the rate of change of the input signal, with the formula: $$u_o = -RC \frac{du_i}{dt}$$ - The faster the input signal changes (e.g., the edge of a square wave), the larger the amplitude of the output $u_o$; - When the input signal is a slowly varying signal, the amplitude of the output $u_o$ will decrease significantly (the differential circuit has high-pass characteristics).

### Analysis of the results:
- **First-order circuit (Fig3.1/3.4)**:
    
    The square wave input is converted to a **triangular-like (ramp) waveform** (typical of first-order RC/RL circuits: the output follows the integral/differential of the square wave).
- **Second-order circuit (Fig3.2/3.5)**:
    
    The output has more "overshoot/undershoot" (compared to first-order), which is characteristic of second-order systems (e.g., RLC circuits) — the response is affected by damping.
- **Full circuit (Fig3.3/3.6)**:
    
    The output is closer to a "stepped square wave" (or more distorted than first-order), reflecting the combined effect of cascaded first/second-order stages.
### Answer about questions
#### Summarizing the integral circuit and differential circuit:
##### 1. Circuit Structure

- **Integrator Circuit**
    
    Capacitor C is connected in the feedback loop (for op-amp-based circuits) or acts as the output terminal (for passive RC circuits).
- **Differentiator Circuit**
    
    Capacitor C is connected in parallel with the input loop (for op-amp-based circuits) or serves as the input terminal (for passive RC circuits).

##### 2. Key Operating Condition

- **Integrator Circuit**
    
    Requires a time constant τ=RC≫T/2 (where T is the period of the input signal).
- **Differentiator Circuit**
    
    Requires a time constant τ=RC≪T/2.

##### 3. Input-Output Relationship (Op-Amp-Based Configurations)

- **Integrator Circuit**
    
    The output is the integral of the input signal:
    
    uo​=−RC1​∫ui​dt
- **Differentiator Circuit**
    
    The output is the derivative of the input signal:
    
    uo​=−RC⋅dtdui​​

##### 4. Waveform Transformation (With Square Wave Input)

- **Integrator Circuit**
    - Ideal output: A linear triangular (ramp) wave.
    - Experimental output: An approximate triangular wave (slope varies with the input signal frequency).
- **Differentiator Circuit**
    - Ideal output: Sharp pulses (only generated at the rising/falling edges of the square wave).
    - Experimental output: Narrow pulses (may include slight oscillation).

##### 5. Frequency Characteristics

- **Integrator Circuit**
    
    Exhibits low-pass behavior: Low-frequency signals pass through easily, while high-frequency signals are suppressed.
- **Differentiator Circuit**
    
    Exhibits high-pass behavior: High-frequency signals pass through easily, while low-frequency signals are suppressed.

#### Three primary factors contribute to the observed deviations:

1. **Manual Cursor Positioning Inaccuracy**
	1. Ambiguous waveform edges (due to noise/ringing) and limited precision of manual cursor placement introduce errors in measuring pulse amplitude, ramp slope, and timing, distorting derivative/integral calculations.
    
2. **Circuit Delay-Induced Spike Offset (Differentiators)**
	1. Parasitic parameters and op-amp bandwidth limits cause signal delays, shifting the differentiator’s output spike relative to the input edge and hindering precise calculations.
    
3. **Waveform Distortion**
	1. Non-ideal components (e.g., resistor/capacitor parameter deviations), parasitic capacitance/inductance, and non-ideal input signals (e.g., square wave edge slope) lead to output waveform distortion (e.g., overshoot, nonlinear ramps), deviating from ideal theoretical waveforms.
